Version control system adalah sebuah system yang merekam perubahan pada file dari waktu ke waktu, sehingga kita bisa melihat versi sebelumnya jika diinginkan

Git adalah salah satu version control system.

Manfaat git :
Bisa menyimpan seluruh versi source code
Berkolaborasi membangun sistem jadi mudah
Bisa berkontribusi di proyek open source
Memudahkan dalam tracking perubahan
Memahami cara deploy modern
